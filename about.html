How to use?

When installing the program, set the installation directory to drive C After installing the program, please add it to 'Start' After the addition is complete, there are two ways to execute it. The first is to enter "hanersc filename. hanersc" in the command prompt, but the prerequisite is to add its directory. If not, please enter its specified directory before executing the file The second option is to run directly, setting the hanersc file opening method to double-click on the installed program at that time to open it directly
